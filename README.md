# Car-sales-analysis
Car Sales Analysis Dashboard is an interactive Power BI project that provides insights into car sales performance and customer preferences. It analyzes top-selling models, transmission types, color trends, and state-wise sales distribution. The dashboard includes dynamic visualizations to identify yearly and monthly sales patterns. It helps businesses make informed, data-driven decisions through effective data analysis and reporting.

🚗 Car Sales Analysis Dashboard – Power BI

📌 Project Overview

This project presents a comprehensive Car Sales Analysis Dashboard developed using Microsoft Power BI. The dashboard provides insights into car sales performance, top-selling models, transmission types, color preferences, geographic sales distribution, and yearly trends.

The objective of this project is to transform raw sales data into meaningful business insights that help stakeholders understand customer preferences and sales patterns.

🎯 Objectives
Analyze car sales performance across different years.
Identify top-performing car models.
Understand customer preferences for transmission types.
Analyze popular vehicle colors.
Evaluate sales distribution across different states.
Provide an executive-level summary for decision-making.

Manual Transmission

4. Color Preference Analysis
Displays the most preferred car colors.

5. Geographic Sales Analysis

📈 Key Insights
Automatic transmission vehicles dominate overall sales.
White and Black are the most preferred vehicle colors.
